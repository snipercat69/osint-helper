# osint-helper

OSINT (Open-Source Intelligence) reconnaissance Discord bot. Supports username enumeration across platforms, domain DNS/RDAP lookups, IP geolocation, email reconnaissance, ASN lookup, and multi-indicator IOC enrichment. Includes rotating proxy support for privacy.

## Topics
python, discord, bot, osint, reconnaissance, cybersecurity, ethical-hacking

## Setup

```bash
# Clone the repo
git clone https://github.com/snipercat69/osint-helper.git
cd osint-helper

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python3 osint-helper.py
```

## About

Built with [OpenClaw](https://github.com/openclaw/openclaw) — an AI agent framework. This app runs as a Discord bot / scheduled automation powered by OpenClaw's cron and channel integration system.
